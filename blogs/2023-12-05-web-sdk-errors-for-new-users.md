Dec 5 , 13:30 UTC Resolved - On 12/5/2023 from 1:36pm UTC to 04:52pm UTC, the Didomi Web SDK failed to fully load on the first page visited by users on a small number of websites with the IAB TCF and Google Additional Consent enabled. The issue was fixed by rolling back to the previous Web SDK version at 04:52pm UTC on 12/5/2023. A post-mortem analysis was done and can be obtained by reaching out to our team.
